# usdglo

## Tasks

### Setup git hooks

`npm run prepare`

### Compile contracts

`npm run compile`

### Running tests

`npm run test`

### Running coverage

`npm run coverage`

### Running tests and report gas

`REPORT_GAS=True npm run test`

### Run slither on contracts

`npm run slither`

### Run prettier on codebase

`npm run fmt`
